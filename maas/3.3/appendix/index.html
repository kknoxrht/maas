Red Hat OpenShift AI: Models as a Service Enablement

Welcome to the Red Hat OpenShift AI Models as a Service enablement workshop!

What you’ll learn

In this workshop, you will:

  • Understand the GPU silo crisis and how Models as a Service (MaaS) solves resource allocation challenges

  • Design and implement a centralized model hub with governed token-based consumption

  • Configure MaaS on Red Hat OpenShift AI using the operator and Custom Resource Definitions

  • Deploy generative AI models using llm-d, vLLM, and KServe components for distributed inference

  • Publish models to the OpenShift AI dashboard catalog for self-service access with role-based permissions

  • Implement governance policies with tiered service limits and rate limiting using Kuadrant

  • Generate usage reports for chargeback and monitor GPU utilization metrics with Grafana

  • Integrate AI models into applications using OpenAI-compatible APIs with proper retry logic

Who this is for

This workshop is designed for Platform Engineers and Developers who want to transform GPU infrastructure from isolated team silos into a centralized, governed environment where lines of business can access generative AI models to build agentic applications.

Experience level: Intermediate

Prerequisites

Before starting this workshop, you should have:

  • Red Hat OpenShift administration experience (Version {ocp_version} or later)

  • Basic Kubernetes operator management knowledge

  • Familiarity with core GPU concepts and generic GPU enablement

  • Understanding of API concepts and REST principles

  • Basic command line experience

Workshop environment

Your lab environment includes a Single Node OpenShift (SNO) cluster with multiple NVIDIA GPUs (minimum {gpu_memory}) pre-configured with:

  • Red Hat OpenShift 4.20 or later

  • Red Hat OpenShift AI 3.4.0 or later (operator installed)

  • GPU operators and drivers configured

  • Required infrastructure components (KServe, Service Mesh)

Estimated time

This workshop will take approximately 3-4 hours to complete.

You can work at your own pace.
